# PAPER_ORC

AI-assisted paper reproduction, evaluation, and experiment orchestration workspace.

## Main workflow

Spec / Paper Idea → Experiment Plan → Python Simulation → Result Analysis → Figures → Paper Draft

## Directory structure

- `paper/` : LaTeX or Markdown paper draft
- `prompts/` : archived AI-agent prompts
- `src/` : Python source code
- `scripts/` : runnable experiment scripts
- `configs/` : experiment configuration files
- `data/` : small reference data only
- `results/` : experiment summaries, CSVs, plots
- `figures/` : paper-ready figures
- `docs/` : technical notes
- `ai_context/` : current project status and agent context
